WELCOME TO GeocodR

An interactive, web-based tool to help you with all your geocoding needs.

GeocodR enables intuitive and rapid geocoding.

 WHAT YOU CAN DO WITH GeocodR:

  • Find coordinates for partial or full addresses.
  • Review your geocoded data in tabular form or via an interactive map. 

 WHAT YOU NEED BEFORE YOU USE GeocodR:

  • A personal API key, get one from the HERE website.
    • Create a freemium project account and obtain a HERE api key - this is your secret password required to use GeocodR. 
    • Freemium projects cap out at 250K geocode transactions per month.  
    • Each address in your uploaded file will count as one transaction.  
  • A DIG key.
    • This is a permission key required to use the tool.  
    • You can obtain this key by contacting Mike Gaunt (email at the bottom of this page).  
    • This key will be changed periodically.  
  • A list of addresses you want geocoded.  
    • Click on the relevant tab to perform your geocoding operation - Addresses (default).  
    • Addresses must be contained in a single file with a single addresses column.  
    • Sample data of various sizes can be found in the project GIT repo.  
    • This application strictly uses the ESPG:4326 cooridante reference system for mapping purposes, all results will be projected with this CRS.
